학생 아바타 이미지 가이드
============================

이 폴더에 학생 이미지를 넣으면 자동으로 적용됩니다.

파일명 규칙:
  {학생ID}.png  →  예: 1.png, 2.png, 3.png ... 25.png

권장 크기: 70x90px 이상 (세로가 더 긴 증명사진 비율)

우선순위:
  1. /avatars/{id}.png  (학생 개별 이미지)
  2. /avatars/default-male.png 또는 default-female.png  (성별 기본 이미지)
  3. CSS 캐릭터  (자동 생성 fallback)

이미지가 없는 학생은 외모 설명을 기반으로 CSS 캐릭터가 자동 생성됩니다.
